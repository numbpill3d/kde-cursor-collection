# KDE Cursor Collection

A collection of high-quality cursor themes for KDE Plasma. These are packaged as .tar.gz files ready for upload to the KDE Store.

## Included Themes

### Gaylence
A futuristic and vibrant rainbow-themed set of animated cursors. Originally created by Gayce.

### Internal Combustion (Animated)
A high-octane cursor set featuring realistic animated fire effects. Originally created by Censor_Not.

### Killers
An edgy and stylized cursor set inspired by the "Natural Born Killers" aesthetic. Originally created by adrenochromedream.

### Lightning Plasma
Electrifying animated cursors styled after the glow of plasma-ball lamps. Originally created by Censor_Not.

### RuneScape Dragon 2H
A nostalgic set of cursors based on the iconic Dragon 2H sword from RuneScape. Originally created by PhilEvil (CC-BY).

### Shadow the Hedgehog
A complete set of cursors featuring sprites of Shadow the Hedgehog from Sonic Battle.

### Sistema Flecha 3D
A professional-grade 3D arrow cursor system with a retro-modern perspective.

### Smoke Weed Everyday
A vibrant and psychedelic "trippy" cursor pack with a meme-inspired aesthetic. Originally created by CatLover3333.

### Spirit Blossom
Elegant animated cursors inspired by the Spirit Blossom skin line from League of Legends. Originally created by Heureux.

## Installation
1. Download the `.tar.gz` for the theme you want.
2. Extract it to `~/.icons/` or use the KDE Plasma Cursor settings to "Install from File".
